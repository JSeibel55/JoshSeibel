Bird Migration Heatmap

This web map originated out of a final project for Geography 573, Advanced Geocomputing and Geospatial Big Data Analytics. The original project scope was meant to use Python to analyze and visualize datasets that are usually too large for more standard GIS technologies. web map was created through a Python script using the package Folium which allows for analysis/agggregation of large datasets and then output the data as time series heatmaps
